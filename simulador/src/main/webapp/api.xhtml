Autocontrol - API
Method Url Params Retorna Descripcion
GET /verify - [ok=200,err=500] Metodo para comprobar si el servicio rest esta funcional
GET /altaCentro/{id}/{gps} [id=id del centro,gps=geolocacion del centro] [ok=200,err=500] Metodo para dar de alta un nuevo centro
GET /altaBateria/{id}/{idCentro}/{gps}/{cm}/{ca} [id=id de la bateria,idCentro=id del centro al que pertenece la bateria,gps=geolocacion del centro,cm=capacidadmaxima en mA,ca=capacidad actual en mA] [ok=200,err=500] Metodo para dar de alta una nueva bateria, con id, id del centro asociado, geolocalizacion, capacidad maxima y capacidad actua
GET /altaTerminal/{id}/{idBateria}/{gps}/{cg} [id=id de la terminal,idBateria=id de la bateria,gps=geolocacion de la terminal,cg=capacidad generadora maxima en mA] [ok=200,err=500] Metodo para dar de alta una nueva terminal, con id, geolocalizacion, capacidad maxima generadora
GET /eventosCentro/{id}/{idUltimoEventoConocido}? [id=id del centro,idUltimoEventoConocido=id del ultimo evento conocido por el cliente] [ok=200,err=500] Metodo para obtener la lista de eventos posteriores al idUltimoEventoConocido del centro dado por id
GET /eventosBateria/{id}/{idUltimoEventoConocido}? [id=id de la bateria,idUltimoEventoConocido=id del ultimo evento conocido por el cliente] [ok=200,err=500] Metodo para obtener la lista de eventos posteriores al idUltimoEventoConocido de la bateria dado por id
GET /eventosTerminal/{id}/{idUltimoEventoConocido}? [id=id de la terminal,idUltimoEventoConocido=id del ultimo evento conocido por el cliente] [ok=200,err=500] Metodo para obtener la lista de eventos posteriores al idUltimoEventoConocido de la terminal dado por id
